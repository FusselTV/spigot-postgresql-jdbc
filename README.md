# Spigot PostgreSQL JDBC

This plugin bundles the PostgreSQL JDBC driver for Spigot/Paper servers.

## Scope

- Supported: Spigot/Paper plugin environments.
- Not supported: Forge.
- Not supported: NeoForge.

## Build

```bash
./gradlew shadowJar
```

Output artifact:

`build/libs/spigot-postgresql-jdbc-1.0.0.jar`
